How It Works

1 Enter your Rust server RCON credentials
2 Type a command — htmx POSTs to /api/rcon
3 Serverless function connects to Rust RCON via WebSocket
4 Response comes back as HTML, htmx swaps it in

Stack

htmx (no JS framework)
Vercel serverless functions
Rust RCON WebSocket protocol
Zero client-side state

Credentials are sent per-request and never stored server-side.
